# Usage

<!-- usage -->
```sh-session
$ npm install -g hoonyc
$ hoonyc COMMAND
running command...
$ hoonyc (-v|--version|version)
hoonyc/0.1.0 darwin-x64 node-v8.11.3
$ hoonyc --help [COMMAND]
USAGE
  $ hoonyc COMMAND
...
```
